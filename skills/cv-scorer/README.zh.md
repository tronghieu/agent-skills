# CV 评估助手 (CV Scorer)

**Language / 语言:** [English](./README.md) | [Tiếng Việt](./README.vi.md) | [中文](./README.zh.md)

此 Skill 用于根据特定的职位描述 (JD) 评估候选人的简历 (CV)，并在 100 分制下提供详细评分和明确的录用建议。

## 此 Skill 有何用途？

CV Scorer 是一款智能简历筛选工具。您无需手动阅读数十份简历来寻找匹配者，只需将职位描述 (JD) 和一份或多份简历输入给 AI。它将分析候选人的经验、技能和资质，并根据他们与岗位的匹配程度输出客观的分数。

## 为什么使用它？

- **节省时间**：快速从大量申请中筛选出合适的候选人。
- **客观评估**：对所有候选人采用统一、无偏见的评分标准。
- **发现潜在问题（红旗）**：自动标记潜在问题，如无解释的职业断档、夸大的数据或重复/矛盾的内容。
- **清晰的建议**：将候选人分为 *推荐*（面试）、*待定*（备选）或 *不匹配*（淘汰）。
- **支持批量处理**：一次性评估多份简历，并自动按照分数从高到低进行排序，方便您选择。

## 工作原理

1. **提供输入**：您输入 **职位描述 (JD)** 和一份或多份 **简历**（文本、Markdown 或 PDF 格式）。
2. **分析 JD**：AI 从 JD 中提取核心要求（必备技能 vs. 加分技能、工作年限、学历要求）。
3. **评分**：AI 根据 5 个维度对简历进行 100 分制评分：
   - **岗位匹配度 (30 分)**：核心技能和工具的契合度。
   - **工作经验 (25 分)**：过往背景和专业级别的相关性。
   - **项目与影响力 (15 分)**：具体成果和数据支撑。
   - **教育与证书 (15 分)**：相关学位或专业资质。
   - **简历排版质量 (15 分)**：布局、清晰度和专业度。
4. **给出建议**：根据分数进行快速分类（推荐：$\ge 70$，待定：$50-69$，不匹配：$< 50$）。

## 如何触发

让 AI 执行如下任务：

```text
根据以下职位描述对这些候选人简历进行评分。
```

```text
筛选此简历并指出其中是否存在任何异常或红旗。
```

```text
评估候选人与软件工程师岗位的匹配度并对他们进行排序。
```
