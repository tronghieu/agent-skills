# System Prompt Creator Skill

**Language / Ngôn ngữ / 语言:** [English](./README.md) | [Tiếng Việt](./README.vi.md) | [中文](./README.zh.md)

This skill guides you through a structured, professional process to create highly effective system prompts (instructions) for any AI model (Claude, ChatGPT, Gemini, etc.).

## What is System Prompt Creator?

A system prompt defines how an AI behaves, what rules it must follow, and how it should format its answers. This skill acts as a professional prompt engineer, helping you design, write, and optimize custom instructions for your AI agents, chatbots, code assistants, or data extraction systems.

## Why Use It?

- **Model-Aware Optimization**: Adjusts prompt formatting specifically for the AI you are using (e.g., XML tags for Claude, strict format parameters for GPT, temperature guidelines for Gemini).
- **Prevents Common Failures**: Helps you build playbooks for complex workflows and write rules that AI won't break.
- **Structured Framework**: Uses a layered architecture (Role, Context, Instructions, Examples, Guardrails) to keep instructions clean and unambiguous.
- **Includes 12 Universal Principles**: Built on official prompt engineering guides from Anthropic, OpenAI, and Google.
- **Saves Engineering Iterations**: Walks you through an interview step to capture all requirements before drafting, reducing back-and-forth edits.

## How It Works

1. **The Interview**: The AI asks you 10 targeted questions to collect details on your target model, use case, persona, tone, rules, and examples.
2. **Analysis**: Classifies prompt complexity (Simple, Moderate, or Complex) to plan sections.
3. **Structured Design**: Uses clean XML tags or clear Markdown headers to separate data from commands.
4. **Drafting with Playbooks**: Employs industry best practices (e.g., explaining the *why* behind rules, defining guardrails, and embedding clear step-by-step playbooks).
5. **Quality Review**: Runs a final check for inconsistencies, over-complicated phrasing, and edge-case handling.

